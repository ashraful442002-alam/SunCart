# ☀️ SunCart – Summer Essentials Store

SunCart is a modern summer-themed eCommerce web application where users can explore and view different summer products like sunglasses, summer outfits, skincare products, beach accessories, and more.

This project was built with Next.js, Tailwind CSS, and DaisyUI with a fully responsive design for mobile, tablet, and desktop devices.

# ✨ Features

- Responsive modern summer-themed UI
- Dynamic Navbar and Footer
- Product listing from JSON data
- Dynamic product details page
- Protected product details route
- User Registration system
- User Login system
- Logout functionality
- My Profile page
- Update Profile feature
- Toast notifications
- Animated UI using Animate.css
- Mobile responsive navbar
- Dynamic routing using Next.js App Router
